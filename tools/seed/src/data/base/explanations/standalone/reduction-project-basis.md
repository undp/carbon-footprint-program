# Datos base del proyecto de reducción

Los campos marcados con un asterisco (\*) son los mínimos necesarios para **guardar un borrador**: el nombre del proyecto, la organización y la huella con reconocimiento de verificación.

El resto de los campos son opcionales mientras el proyecto es un borrador, pero **todos los campos son obligatorios para postular el proyecto a reconocimiento de reducción**.

La única excepción es la casilla **«Reportado en otra iniciativa»**: marcarla es opcional, y su descripción solo es obligatoria cuando la casilla está marcada.

Completa la información faltante antes de postular.
