# Changelog

All notable changes to Huella Latam are documented in this file.

The format is based on
[Keep a Changelog 1.1.0](https://keepachangelog.com/en/1.1.0/), and
this project adheres to
[Semantic Versioning 2.0.0](https://semver.org/spec/v2.0.0.html). See
[`docs/release/versioning.md`](./docs/release/versioning.md) for the
project's release governance.

Country deployments are independent and may be on different upstream
versions. Adopting governments are responsible for tracking the
changes between the version they run and the latest upstream release.

## [Unreleased]

### Added

- `LICENSE` — MIT License with copyright held by the United Nations
  Development Programme (UNDP) and a reservation clause covering the
  use of the UN name and emblem.
- `NOTICE` — attribution to UNDP Project 01000983 (Climate Hub) and
  the UNDP Climate Promise initiative.
- `AUTHORS` and `MAINTAINERS.md` — public statement of project
  ownership and the active maintainer team.
- `.github/CODEOWNERS` — path-based pull-request review routing.
- Root `CONTRIBUTING.md` — short, DPG-aligned entry point that links
  to the full guide at `docs/development/contributing.md`.
- `CODE_OF_CONDUCT.md` — community standards based on the Contributor
  Covenant 2.1.
- `SECURITY.md` — responsible-disclosure policy with GitHub Private
  Vulnerability Reporting as the primary channel and an email
  fallback.
- `PRIVACY.md` — public-facing privacy notice covering data inventory,
  legal basis, retention, data-subject rights and country variants.
- `docs/overview/sdg-alignment.md` — mapping to the UN 2030 Agenda
  (primary: SDG 13; secondary: SDG 12, 17, 11, 9). Provides DPGA
  Standard v1.1.6 Indicator 1 evidence.
- `docs/governance/` — new subfolder for DPG-compliance artefacts.
- `docs/governance/principles-for-digital-development.md` — mapping
  of the nine Principles for Digital Development to concrete files
  and practices in the codebase. Provides DPGA Standard v1.1.6
  Indicator 8 evidence.
- `docs/governance/acceptable-use.md` — content policy with
  prohibited-content categories, current and roadmap detection
  controls, report-and-takedown workflow and the country-addendum
  framework. Provides DPGA Standard v1.1.6 Indicator 9B evidence.
- GitHub issue and pull-request templates under `.github/`,
  including a `dpg_compliance.yml` template for reporting gaps
  against the DPGA Standard.
